Aprendendo a usar "Property binding" e "Two-way data binding"


"Property binding" é definir o valor de um atributo de uma tag, por expressao.
Obs: Nos exemplo abaixo, estou usando uma expressao para habilitar e desabilitar os botoes, baseados o valor deu uma variavel.

"Two-way data binding" é usar o atributo especial [(ngModel)] para conectar o elemento com uma variavel no arquivo 'app.component.ts'.
